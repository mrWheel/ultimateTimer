WiFi Manager Screen

This screen appears when the device opens its WiFi configuration portal.

Manual Home Timer Screen Main Menu Cyclic Timer 24h Timer WiFi & Web UI

When it appears

  • When there are no working WiFi station credentials.
  • After repeated connection failures.
  • After you select Start WiFi Manager in System Settings.

How to use it

  1. Connect your phone or laptop to the shown access point.
  2. Open the shown IP address if the captive portal does not open automatically.
  3. Choose your local WiFi network and enter its password.
  4. After credentials are accepted, the device stores them and restarts.

The fallback access point in this repository has no password.

Cancel behavior

Select Cancel WiFi Manager to leave WiFi Manager, keep WiFi disabled for now, and return to the Timer Screen.

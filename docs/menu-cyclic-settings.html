Cyclic Timer Settings Menu

Use this menu to edit the cyclic timer values that control repeated ON/OFF timing.

What you can edit

  • On Time / Off Time: numeric field input.
  • On Time Unit / Off Time Unit: choose ms, s, or Min.
  • Number of Cycles: zero means infinite cycles.
  • Trigger Mode: Manual or External.
  • Trigger Edge: Rising or Falling.
  • Lock Input During Run: block trigger/reset changes while the timer is busy.

Editing rules

Field input example
Title: On Time
Value: 0005
Turn  = change current digit
Short = move right
K0    = move left
Hold  = save and return
  • Short press the encoder to enter the selected field.
  • Encoder medium/long press saves the current field and returns.
  • PIN_KEY0 short moves left one position.
  • PIN_KEY0 medium/long saves and returns.

Saving behavior

  • Changes always affect the running timer immediately.
  • If Auto Save Profile is Yes, the active profile is updated immediately too.
  • If Auto Save Profile is No, use Save Profile later to keep the new values.

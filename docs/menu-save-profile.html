How it works

  • The menu always targets the current active profile.
  • The default selection is No to avoid accidental saves.
  • Select Yes to save and return to the Main Menu.
  • Select No to cancel and return without saving.

When you should use it

Use this menu whenever Auto Save Profile is set to No and you want to keep the timer settings you just changed.

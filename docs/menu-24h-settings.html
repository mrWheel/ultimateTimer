24h Timer Settings Menu

Use this menu to build or change the daily 24-hour schedule one hour and one quarter at a time.

Manual Home Timer Screen Main Menu Cyclic Timer 24h Timer WiFi & Web UI

Focus flow

  1. Select the hour.
  2. Select the hour type.
  3. If the hour is split, select the quarter slot.
  4. Set the quarter type.

Hour focus

24h Timer
+--------------------------------+
| Hour                Type       |
| >05<                [R]        |
+--------------------------------+
Turn=Select hour   Short=Lock
Hold=Save+Back     K0=Back

If the hour type is S, the quarter blocks are shown immediately.

Split-hour editing

24h Timer
+--------------------------------+
| Hour                Type       |
| [05]                >S<        |
+--------------------------------+
| [00-15] [16-30] [31-45] [46-59]|
| [-]     [+]     [R]     [-]    |
| Turn=Type   Short=Edit quarters|
+--------------------------------+

>00-15< [16-30] [31-45] [46-59]
[-]     [+]     [R]     [-]
  • Choosing +, -, R, or r applies that value to the full hour.
  • Choosing S unlocks the 4 quarter blocks.
  • Quarter blocks wrap within the hour and use the same short-press confirm behavior.

How to finish

  • Encoder medium press or long press saves all current 24h edits and returns to the Main Menu.
  • PIN_KEY0 medium or long does the same.

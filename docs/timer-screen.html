Timer Screen

The Timer Screen is the home screen. It shows the live timer state and is the only screen you see during normal operation.

Manual Home Timer Screen Main Menu Cyclic Timer 24h Timer WiFi & Web UI

How to open it

The Timer Screen appears automatically at startup and after you leave menus.

Cyclic timer view

Timer Screen
+--------------------------------+
| State              Running     |
| On Time            5 s         |
| Off Time           5 s         |
| Output             ON  004:22  |
| Cycles             3/INF       |
|                                |
| [Start] [Stop] [Reset]         |
+--------------------------------+
  • State shows Idle, Running, Paused, or Finished.
  • Output also shows the countdown to the next output change in MMM:SS.
  • Cycles shows the current cycle and the total cycle limit. INF means infinite cycles.
  • In manual trigger mode you can select Start, Stop, or Reset with the encoder and confirm with a short press.

24h timer view

Timer Screen
+--------------------------------+
| State              Running     |
| Next Change Between 18:16-18:30|
| Last Change        ON  17:45   |
| Next Change        OFF 18:24   |
| Output             ON  038:00  |
|                                |
|        24h auto mode           |
+--------------------------------+
  • Next Change Between shows the possible window for the next scheduled change.
  • Last Change and Next Change show the last and next output transitions.
  • There are no Start/Stop/Reset buttons in 24h mode.

Header information

  • The right side of the header shows No WiFi when the device is offline.
  • When WiFi and time sync are available, the header shows the current time.
  • When Warp Speed is enabled, the header keeps showing the warped clock for cyclic timers, including minute-based cyclic profiles.

Open the menu

Press and hold the encoder to open the Main Menu.

Cyclic Timer

The cyclic timer repeats an ON phase and an OFF phase until the cycle count is finished or forever when the repeat count is zero.

Manual Home Timer Screen Main Menu Cyclic Timer 24h Timer WiFi & Web UI

What it is for

  • Use it when you need repeating ON/OFF output timing.
  • You can define the ON time, OFF time, units, number of cycles, trigger mode, and trigger edge.
  • repeatCount = 0 means run forever.

Main fields

On Time
Duration of the ON phase.
Off Time
Duration of the OFF phase.
Units
Each time can use ms, s, or Min.
Number of Cycles
How many ON/OFF repetitions to run. Zero means infinite.
Trigger Mode
Manual shows Start/Stop/Reset buttons. External waits for the trigger input.
Trigger Edge
Select Rising or Falling for the external trigger input.

Manual and external trigger behavior

Manual trigger mode
[Start] [Stop] [Reset]

External trigger mode
External Trigger
  • In Manual mode, start and stop the timer from the Timer Screen or the web page.
  • In External mode, the Timer Screen hides the action buttons and shows External Trigger.

Warp Speed

  • Warp Speed is only for cyclic timers.
  • When enabled, one real second advances the cyclic runtime by 60 seconds for phases using s or Min.
  • If either cyclic unit is set to Min, warp acceleration remains active across both phases.
  • The countdown still looks normal in MMM:SS format.
  • The Cycles tile also shows Warp mode.
  • Warp Speed is forced back to Disabled on every reboot.

Where to edit it

Use the Cyclic Timer Settings Menu on the TFT display or the Web UI.

What each setting does

SSID / IP / MAC / Firmware
Read-only information.
Encoder Order
Reverse encoder direction if left/right feels wrong.
Warp Speed
Enable or disable 60x time acceleration, including minute-based cyclic profiles.
Display Rotation
Switch between rotation 1 and rotation 3.
Erase WiFi credentials
Clear saved station WiFi name and password.
Start WiFi Manager
Launch the WiFi setup portal and restart into WiFi-enabled mode.
Output Polarity
Choose Active High or Active Low output behavior.
Auto Save Profile
Choose whether timer-setting changes also update the active profile automatically.
Theme Color
Choose Red, Green, Blue, Indigo, Violet, or Yellow.
Restart ultimateTimer
Restart the device after confirmation.

Persistence

These settings are stored in device preferences, not in individual timer profile files.

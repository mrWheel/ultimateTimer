WiFi and Web UI

When WiFi is available, Ultimate Timer also provides a browser-based control panel.

Manual Home Timer Screen Main Menu Cyclic Timer 24h Timer WiFi & Web UI

How to open it

  • After the device connects to WiFi, the serial monitor prints the IP address.
  • Open http://<device-ip>/ in a browser on the same network.
  • If station WiFi is unavailable, use the WiFi Manager Screen first.

Page structure

Top bar: Active profile   12:34:56

System | Timer Settings | Profiles

Timer Screen  (always visible)
+--------------------------------+
| State, output, countdown       |
| Cycles or 24h timing details   |
| [Start] [Stop] [Reset]         |
+--------------------------------+
  • System contains device-wide settings.
  • Timer Settings opens Cyclic Timer Settings and 24h Timer Settings.
  • Profiles opens Load, Save, New, and Delete Profile pages.

Important web behavior

  • Opening any web menu tile automatically stops the timer.
  • Closing all menu tiles automatically starts the timer again.
  • The Timer Screen tile is always visible and keeps refreshing.
  • When timer settings differ from the saved profile, the header adds (not saved) after the profile name.
  • For 24h profiles, the Start/Stop/Reset buttons are hidden.

Saving rules

  • Timer Settings use live apply while you edit.
  • System settings only change when you press Save.
  • If Auto Save Profile is No, saving timer settings shows a warning because the running state changed but the profile file did not.
  • The Save Profile button stays available even while the timer is running.

24h editor in the browser

Hour | Type | 00-15 | 16-30 | 31-45 | 46-59
05   |  S   |   -   |   +   |   R   |   -
06   |  +   | locked| locked| locked| locked

Selecting -, +, R, or r applies that state to the full hour. Select S to unlock the quarter cells.

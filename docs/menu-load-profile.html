How it works

  • Rotate the encoder to choose a profile, then short press to load it.
  • The list always includes the protected built-in profiles default and default-24h.
  • Loading a profile resets the executed cycle counter to zero.
  • Loading a 24h profile immediately makes the output follow the current wall clock position.

Exit option

Select Exit or use PIN_KEY0 medium/long to return without loading anything.

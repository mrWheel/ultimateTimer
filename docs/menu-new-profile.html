New Profile Menu

Use this menu to create a new profile name directly from the rotary encoder.

New Profile Menu
Profile: Abc-----
          ^
Turn  = change token
Short = move right
K0    = move left
Hold  = save name

How naming works

  • The name is entered one position at a time.
  • Available characters include letters, digits, and the dash.
  • If the current character is -, a short press does not move to the right.
  • When the current timer type is 24h, the file is stored with an automatic -24h suffix.

Saving the new name

  • Encoder medium/long press saves the new profile name and returns.
  • PIN_KEY0 medium/long also saves and returns.
  • PIN_KEY0 short on the first position exits without saving.

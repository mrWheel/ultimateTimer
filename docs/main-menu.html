Main Menu

The Main Menu is the entry point for all local configuration tasks.

Manual Home Timer Screen Main Menu Cyclic Timer 24h Timer WiFi & Web UI

How to open it

Press and hold the encoder while the Timer Screen is visible.

Menu items

Cyclic Timer Settings
Edit ON/OFF durations, units, cycles, trigger mode, and trigger edge.
24h Timer Settings
Edit the quarter-hour schedule for the 24h timer.
Save Profile
Write the current timer settings to the active profile.
Load Profile
Load another saved profile.
New Profile
Create a new profile name with character-by-character entry.
Delete Profile
Delete a profile after confirmation.
System Settings
Change device-wide settings such as theme, output polarity, and WiFi actions.
Exit
Return to the Timer Screen.

Navigation rules

  • Rotate the encoder to move up and down.
  • Short press the encoder to open the selected item.
  • The menu does not wrap around at the top or bottom.
  • A medium or long press on PIN_KEY0 acts like Back/Exit.
